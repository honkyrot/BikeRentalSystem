# BikeRentalSystem

## Overview
A small application that keeps track of bike rentals.<br>
Created for a final project for school.

## Features
- A GUI for ease of use.
- Keep track of bikes rented.
- Calculate rental fees based on hourly rates.
- Generate rental reports.

## how to run
install required packages from `requirements.txt`<br>
start `src/gui_controller.py`
```bash
pip install -r requirements.txt
python src/gui_controller.py
```
